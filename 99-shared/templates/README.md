# Templates (Reusable)

วาง template ที่ใช้ซ้ำได้ที่นี่ — ทุก domain สามารถเรียกใช้ได้

## แนะนำให้สร้าง
- `gov-proposal-template.docx` — Template โครงการรัฐ
- `gov-budget-template.xlsx` — Template งบประมาณ
- `trading-strategy-template.md` — Template เขียนกลยุทธ์
- `backtest-report-template.xlsx` — Template รายงาน backtest
- `n8n-workflow-template.json` — Template n8n boilerplate

ขอให้ Claude สร้างเพิ่มได้ทุกเมื่อ — บอกว่า "สร้าง template สำหรับ X"
