# 03 — Government Planning & Budgeting

โครงการ / งบประมาณ / นโยบาย / รายงานหน่วยงานรัฐ

## โครงสร้าง
- `proposals/` — โครงการ / ข้อเสนอ (docx)
- `budgets/` — งบประมาณรายปี / รายโครงการ (xlsx)
- `policies/` — นโยบาย / ระเบียบ / คู่มือ
- `reports/` — รายงานผลการดำเนินงาน, KPI

## Naming Convention
```
[ปีงบ]_[หน่วยงาน]_[ชื่อโครงการ].docx
ตัวอย่าง: 2569_สพฐ_พัฒนาทักษะดิจิทัลครู.docx
```

## Template โครงการมาตรฐาน
1. ชื่อโครงการ
2. หลักการและเหตุผล
3. วัตถุประสงค์ (SMART)
4. กลุ่มเป้าหมาย
5. วิธีดำเนินการ / ปฏิทินงาน
6. งบประมาณ (รายละเอียดหมวด)
7. ตัวชี้วัด KPI / ประเมินผล
8. ผู้รับผิดชอบ
